BRC Bookmarklet
Installer

Install this once. Then tap it on any Yahoo BRC standings page — logged in on any device — and it auto-downloads the .js import file.

Step 01

Make sure your bookmarks bar is visible

In Chrome: Ctrl+Shift+B  ·  In Firefox: Ctrl+Shift+B  ·  In Safari: View → Show Favorites Bar

Step 02

Drag this button to your bookmarks bar

Click and drag the button below directly onto your browser's bookmarks bar. That's it — it's installed.

🏀 BRC Import
Drag me to the bookmarks bar.
Don't click — drag. If you click it now it'll run on this page (harmlessly).

✓ How to use it

1
Go to your Yahoo BRC standings page while logged in.
2
Click the 🏀 BRC Import bookmark in your bar.
3
A .js file downloads automatically. Load it into ofl-data.html via Import Config.
Step 01

Copy the bookmarklet code

Tap the button below to copy the full bookmarklet code to your clipboard.


✓ Copied!
Step 02

Save any page as a bookmark

In Safari: tap the Share button → Add Bookmark. Name it 🏀 BRC Import and save it anywhere.

In Chrome (mobile): tap ⋮ → Bookmark this page. Name it 🏀 BRC Import.

Step 03

Replace the URL with the copied code

In Safari: tap the address bar → tap Bookmarks → find 🏀 BRC Import → tap Edit → clear the URL field and paste the copied code.

In Chrome (mobile): tap ⋮ → Bookmarks → long-press 🏀 BRC Import → Edit → clear the URL and paste.

⚠ Make sure the entire code starting with javascript: is pasted — don't leave any of the old URL behind.

✓ How to use it

1
Go to your Yahoo BRC standings page while logged in.
2
Tap the address bar and type BRC — the bookmark will appear. Tap it.
3
A .js file downloads (or a share sheet appears). Load it into ofl-data.html via Import Config.
📱 On iOS Safari, downloads go to Files → Downloads. You may get a share sheet instead of an auto-download — tap Save to Files.
