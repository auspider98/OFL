✓ How to use it

1
Go to a bracket page on Yahoo while logged in.
2
Tap the address bar, type BRC, tap 🏀 BRC Picks.
3
File downloads. Repeat for all 13 brackets.
📱 iOS: files go to Files → Downloads. You may get a share sheet — tap Save to Files.
What each .js file contains
PICKSEvery team picked to win per round (R64 → Final), with seed number and slot ID, across all 4 regions
STATUSPer-pick status: correct, incorrect, or pending (game not yet played)
RESULTSActual game results extracted from the same page — winner, loser, both seeds, scores for every completed game
CHAMPIONNational champion pick from the bracket's final championship slot
MEMBER IDBracket name matched to OFL member ID via the hardcoded roster map
SUMMARYQuick stats: total picks, correct count, incorrect count, pending count
